# 🏥 Digital Prescription & Hospital Management System

## 📌 Overview
This project is a **Web-Based Hospital Management System with Digital Prescription and Medical Record Management** developed using Java and Spring Boot. It provides a platform to manage patients, doctors, prescriptions, and payments efficiently, replacing traditional paper-based methods.

---

## 🎯 Features

### 👨‍💼 Admin Module
- Add and manage doctor details  
- View doctor records  
- Control system operations  

### 👨‍⚕️ Doctor Module
- Add patient details  
- View and manage appointments  
- Generate digital prescriptions  
- Manage payments  

### 👤 Patient Module
- Login and access dashboard  
- Request appointments  
- View prescriptions  
- Make payments  
- Download prescription as PDF  

---

## ⚙️ Technologies Used
- **Frontend:** HTML, CSS, Thymeleaf  
- **Backend:** Java, Spring Boot  
- **Database:** MySQL  
- **Tools:** VS Code / Eclipse, MySQL Workbench

## 🔄 System Workflow
1. User logs in (Admin / Doctor / Patient)  
2. System verifies credentials  
3. Role-based dashboard is displayed  
4. Doctor manages patients and prescriptions  
5. Patient requests appointment and makes payment  
6. Prescription is generated and downloaded as PDF  
7. Data is stored in MySQL database  

---

## 🛠️ Installation & Setup

1. Clone the repository:

2. Open the project in your IDE (Eclipse / VS Code)

3. Configure MySQL database:
- Create a database
- Update `application.properties` with DB credentials

4. Run the Spring Boot application

5. Open browser and go to:
http://localhost:8080
